ReVi is a cross-platform terminal based Vim inspired text editor.

# Useage

```sh
git clone https://github.com/revi-editor/revi
cd revi
cargo install --path .
revi <filename>
```

# Development Use
```sh
git clone https://github.com/revi-editor/revi
cd revi
cargo run --release -- <filename>
```
